Features of Stratum
Providing you the best features
Levelling

A prized feature of Stratum, the levelling and xp system will set your server far apart from others. The levelling system includes customizable levels (assign levels to roles), leaderboards and rankings, and further individual stats, all of which are earned just by chatting in your server! Anyone in your server can gain xp by chatting, and it will keep the conversations going in your server! Combined with Stratum's upcoming auto-moderation, this is the ideal bot.

Management

Stratum is incredibly modular. Use its advanced settings to customize it to your server's needs! This includes Admin S.T.A.P.L.E.S like enabling and disabling commands, server-logging, case-logging, moderation commands, and so on. One way to customize your server is by using the settings command. Use it to turn on features like levelling roles, logging, and much more.

Games/Fun

Stratum is always gaining new fun commands and game features. Spice up your server with many fun and entertaining commands with jokes, trivia, would you rather, image generation, and more.
